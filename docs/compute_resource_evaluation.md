## 💡 優化建議

### 訓練優化

1. **Mixed Precision (FP16)**
   - 可減少 ~40% VRAM 使用
   - 加速 ~30% 訓練速度
   - 在 `finetune_bert.py` 中加入 `fp16=True`

2. **Gradient Accumulation**
   - 允許更大 effective batch size
   - 當 VRAM 不足時使用

3. **使用單 GPU**
   - 目前有 GPU 不平衡警告
   - 建議設定 `CUDA_VISIBLE_DEVICES=0` 只用 4090

### LIME 優化

1. **減少 num_samples**
   - 可從 1000 降至 200-500
   - 對結果影響不大，速度提升明顯

2. **Batch 處理**
   - 目前 batch_size=16
   - 可提升至 32 加速推論
